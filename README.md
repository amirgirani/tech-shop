# tech-shop
🛒 TechShop (تک‌شاپ) - فروشگاه آنلاین گجت‌های هوشمند

تک‌شاپ یک وب‌اپلیکیشن تک‌صفحه‌ای (SPA) مدرن، سریع و تعاملی برای فروش گجت‌های هوشمند، لپ‌تاپ و تجهیزات گیمینگ است. این پروژه به عنوان یک نمونه‌کار (Portfolio) حرفه‌ای طراحی شده و تمرکز آن بر روی تجربه کاربری (UX) روان، انیمیشن‌های جذاب و معماری تمیزِ فرانت‌اند است.

✨ ویژگی‌های کلیدی

معماری SPA (وب‌اپلیکیشن تک‌صفحه‌ای): جابجایی بین صفحات (خانه، فروشگاه، سبد خرید، درباره ما، تماس با ما) بدون رفرش شدن صفحه و با بالاترین سرعت.

سبد خرید داینامیک: مدیریت کامل سبد خرید (افزودن، کاهش/افزایش تعداد، حذف) با محاسبه لحظه‌ای مالیات و جمع کل.

ذخیره‌سازی محلی (Local Storage): حفظ اطلاعات سبد خرید کاربر حتی پس از بستن مرورگر.

دستیار هوشمند (AI Chatbot UI): شبیه‌ساز چت‌بات هوش مصنوعی با قابلیت پاسخگویی خودکار و ایمن‌سازی شده در برابر حملات تزریق کد (XSS).

انیمیشن‌های روان (Scroll Reveal): استفاده از Intersection Observer API برای نمایش زیبای المان‌ها همگام با اسکرول کاربر.

طراحی کاملاً واکنش‌گرا (Responsive): سازگاری کامل با موبایل، تبلت و دسکتاپ.

سئوی داینامیک (Dynamic SEO): تغییر خودکار Title و Meta Description با جاوااسکریپت هنگام تغییر صفحات در حالت SPA.

سیستم اطلاع‌رسانی (Toast Notifications): نمایش پیام‌های موفقیت‌آمیز به صورت پاپ‌آپ‌های محو شونده.

🛠 تکنولوژی‌های استفاده شده

HTML5: ساختار معنایی (Semantic)

CSS3 / Bootstrap 5 (RTL): استایل‌دهی، سیستم گرید و واکنش‌گرایی

JavaScript (Vanilla - ES6+): منطق برنامه، مسیریابی (Routing) و مدیریت State

Bootstrap Icons: آیکون‌های وکتور

Vazirmatn (وزیرمتن): تایپوگرافی استاندارد وب فارسی

🛡 امنیت و بهینه‌سازی (Security & Best Practices)

در توسعه این پروژه، به اصول امنیتی فرانت‌اند توجه ویژه‌ای شده است:

جلوگیری از حملات XSS: تمامی ورودی‌های کاربر (مانند جستجوی محصولات و فرم چت‌بات) پیش از رندر شدن در DOM، پاکسازی (Sanitize) می‌شوند تا از اجرای کدهای مخرب جلوگیری شود.

کپسوله‌سازی منطق (Logic Encapsulation): توابع جاوااسکریپت به گونه‌ای نوشته شده‌اند که کمترین تداخل را با Scope جهانی (Global Window) داشته باشند.

بهینه‌سازی انیمیشن‌ها: انیمیشن‌های اسکرول به جای رویداد سنگین onscroll، با Intersection Observer پیاده‌سازی شده‌اند که تاثیر مستقیمی بر روی فریم‌ریت (FPS) و عدم افت سرعت مرورگر دارد.

🚀 راهنمای اجرا

از آنجایی که این پروژه کاملاً در سمت کاربر (Client-Side) اجرا می‌شود و فعلاً نیازی به دیتابیس یا سرور بک‌اند ندارد، اجرای آن بسیار ساده است:

پروژه را Clone یا دانلود کنید.

فایل index.html (یا TechShop_App.html) را با هر مرورگر مدرنی (ترجیحاً Chrome یا Firefox) باز کنید.

برای تجربه بهتر در زمان توسعه، می‌توانید از افزونه Live Server در VS Code استفاده کنید.
